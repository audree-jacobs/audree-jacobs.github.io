Typography

Heading Font: Raleway

This is a simple sans-serif font with fun overlapping W letters that add a little bit of interest, especially on a site that has a lot of W words. It is not a very decorative font however so it shouldn't be too distracting.

Body Font: Bitter

This font is a serif font which is typical of body fonts as it is similar to text you would find in a book and is farmiliar to read. It is very simple and legible which is a critical charecterstic of body text.
